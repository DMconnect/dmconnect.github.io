DXP-0003: Ignore

DXP-0003 добавляет функциональность игнорирования пользователей в DMconnect.

Описание:
Расширение позволяет пользователю управлять собственным списком игнорирования. Сообщения от пользователей, находящихся в ignore-листе, автоматически отфильтровываются и не доставляются получателю.

Список игнорирования хранится на сервере и не сбрасывается после перезахода в аккаунт.

Доступные команды:
  • /ignore-add <username> — добавить пользователя в список игнорирования.
    Пример: /ignore-add Bob
  • /ignore-del <username> — удалить пользователя из списка игнорирования.
    Пример: /ignore-del Bob
  • /ignore-list — показать текущий список игнорируемых пользователей.
  • /ignore-clear — полностью очистить список игнорирования.

Ответы сервера:
  • Please log in first.
    Возвращается при попытке использования любой команды расширения без активной авторизации.
  • Usage: /ignore-add <username>
    Возвращается, если команда /ignore-add вызвана без указания имени пользователя.
  • Ignoring <username>.
    Пользователь успешно добавлен в список игнорирования. Если пользователь уже находился в списке, сообщение всё равно возвращается.
  • Usage: /ignore-del <username>
    Возвращается, если команда /ignore-del вызвана без указания имени пользователя.
  • <username> is not in your ignore list.
    Возвращается при попытке удалить пользователя, которого нет в списке игнорирования.
  • Stopped ignoring <username>.
    Пользователь успешно удалён из списка игнорирования.
  • Ignore list: user1, user2, ...
    Возвращается командой /ignore-list, если список игнорирования не пуст.
  • Ignore list is empty.
    Возвращается командой /ignore-list, если список игнорирования пуст.
  • Ignore list cleared.
    Возвращается после успешного выполнения команды /ignore-clear, если список ранее содержал элементы.
  • Ignore list is already empty.
    Возвращается при выполнении /ignore-clear, если список игнорирования уже пуст.
